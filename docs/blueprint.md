# **App Name**: UCCOS ADMIN TOOL

## Core Features:

- Blank Page Display: Display a completely blank HTML page with basic CSS styling.
- Basic Styling: Include a simple CSS reset and some basic body styling.
- JS Inclusion: Ensure the page is ready for Javascript interactions.

## Style Guidelines:

- Primary color: Light gray (#F0F0F0) for a clean, neutral background.
- Background color: Very light gray (#FAFAFA), close to white but providing a subtle contrast.
- Accent color: Soft blue (#A0B4C7), used sparingly for interactive elements if any are added later.
- Font: 'Inter' (sans-serif) for a clean and modern look, suitable for any text added later.
- Simple and clean layout with plenty of whitespace. No initial elements other than the basic HTML structure.
